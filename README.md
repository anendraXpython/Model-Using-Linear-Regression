# 📈 Linear Regression Analysis

A beginner-friendly Machine Learning project that demonstrates **Exploratory Data Analysis (EDA)** and implementation of **Simple & Multiple Linear Regression Models** using Python.

This project helps in understanding how linear regression is used to analyze relationships between variables and make predictions from data.

---

# 🧠 About Linear Regression

Linear Regression is a **supervised machine learning algorithm** used for predicting continuous numerical values.

It finds the relationship between dependent and independent variables by fitting the best possible straight line through the data points.

This project includes:

- Exploratory Data Analysis (EDA)
- Data visualization
- Simple Linear Regression
- Multiple Linear Regression
- Model evaluation and prediction

---

# 🚀 Features

- Perform Exploratory Data Analysis (EDA)
- Visualize relationships between variables
- Build Simple Linear Regression model
- Build Multiple Linear Regression model
- Predict values using trained models
- Beginner-friendly implementation
